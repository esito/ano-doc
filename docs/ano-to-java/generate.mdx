# Generate java Application using DBmasker

:::tip Goal
Use the DBmasker Service to generate the java application that will run your anonymization tasks and rules in the `.ano` file.

:::

## Process

### 1. Log in

[Log in to the DBmasker Dashboard](https://anonymizer.esito.no/auth/dashboard/home)

![alt text](/img/docs/ano-dashboard-wo-dberase.png 'DBmasker Dashboard')

### 2. Navigate to DBmasker

![alt text](/img/docs/dbmasker-dashboard.png 'DBmasker Dashboard')

### 3. Root package

Name the root package as you wish. Default is `example.anonymizer`

### 4. Click **Select A File**, and select your `.ano` file

### 5. Click **Download Zip** to get your Java application

![alt text](/img/docs/dbmasker-generated-java.png 'DBmasker Dashboard')
