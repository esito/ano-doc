# From ANO to Execution using DBmasker

:::tip Goal
Once satisfied with the anonymization rules in the `.ano` file, you will want to execute them against the database.
To do this, follow these steps:

1.  [Generate the Application using DBmasker](.././generate)
2.  [Configure the Application](.././configure)
3.  [Execute the Appliation](.././execute)

:::

## Requirements for executing the Java Application

The generated application currently runs on

- Java 1.8 (Java 8)
- Maven 3.6.1

You will also need to edit yor database connection settings.
