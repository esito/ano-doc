# Generate java Application using DBmasker

:::tip Goal
Use the DBmasker Service to generate the java application that will run your anonymization tasks and rules in the `.ano` file.

:::

1. [Log in to the Anonymizer Dashboard](https://anonymizer.esito.no/auth/dashboard/home)

![alt text](/img/docs/ano-dashboard-wo-dberase.png 'Anonymizer Dashboard')

2. Navigate to DBmasker

![alt text](/img/docs/dbmasker-dashboard.png 'Anonymizer Dashboard')

3. Click **Select A File**, and select your `.ano` file

4. Click **Download Zip** to get your Java application
