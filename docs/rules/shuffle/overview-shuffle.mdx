# The `shuffle` anonymization rule

:::tip Goal
Write `shuffle` rule logic to

- Shuffle the values of a column

:::

The shuffle function reorders the column values in a random order and ensures that no record retains the original column value.

## Code structure

```ano
        ...
        shuffle <column> <rule-name?>
        ...
```
