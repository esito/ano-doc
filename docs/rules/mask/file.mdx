<TabItem value="update">

```ano
table CUSTOMER
    ...
    column text NAME size 40
    ...
...
task ... {
    update CUSTOMER
        ...
        mask NAME
            format "%s %s"
            file src/main/resources/firstname.txt random-order
            file src/main/resources/lastname.txt random-order
        ...
}
```

</TabItem>

<TabItem value="create">

```ano
table CUSTOMER
    ...
    column time EARLIESTCHECKINTIME
    ...
...
task ... {
    update CUSTOMER
        minimum-rows 50
        ...
        mask NAME
            format "%s %s"
            file src/main/resources/firstname.txt random-order
            file src/main/resources/lastname.txt random-order
        ...
}
```

</TabItem>

</Tabs>

- Make sure the `.txt` files resides in the folder `src/main/resources/` in the generated project.
- `random-order`: Optional keyword that decides if values are picked randomly or sequentially from the file.
- With multiple input sources, the order of inputs corresponds to the order of the `%s`Â´s in the `format` string
  - 1st `%s` of the format string recieves values from `<file-1-location>`, 2nd `%s` recieves from `<file-2-location>`
