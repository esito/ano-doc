# Generated Random Values - Sources

## Random Types

The **random-[type]** input source is used to insert a value drawn from a random sequence as defined by the **from** and **to** properties. The **random-[type]** is one of

- `random-integer`
- `random-decimal`
- `random-time`
- `random-date`
- `random-datetime`
