# `random-decimal`

:::tip Goal
Write `mask` rule logic to

- generate **random decimals** from an interval defined by `<from>` and `<to>` properties

:::

## Code structure

```ano
        ...
        mask <column> <rule-name>
            format %f
            random-decimal <from> <to>      // [<from>, <to>]
        ...
```

- Specify desired format output format
- `<from>` and `<to>` values are included in the generated values.

    ...

    update HOTELROOMCATEGORY

        mask ACTUALPRICE HOTELROOMCATEGORY_ACTUALPRICE

    ...

## Formatting

If desired result is different than decimal, simply change the format string.
